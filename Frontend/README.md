# Twirrer | Frontend

Using 'React.js' + 'React Hooks' + 'React Context api' to build the frontend of Twirrer.

## Live Demo
[twirrer-app.web.app](https://twirrer-app.web.app/)

## How to run the app locally?

### 1- Install packages
run `npm install`

### 2- Run project
run `npm start`

### 3- Open it
go to `http://localhost:3000`
